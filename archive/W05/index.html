A tool for visualizing variable fonts with axises with p5js

This tool rewrites the way type designers interact with fonts, turning the process into something almost physical. Built on p5js, it connects the subtleties of font adjustments to the movements of a mouse, making variable fonts less about numbers and more about intuition. It’s a hands-on experiment where typography gets as flexible as the imagination allows
